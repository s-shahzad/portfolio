# Portfolio React

Premium React + TypeScript portfolio built from your existing content.

## Stack
- Vite + React + TypeScript
- Tailwind CSS
- Framer Motion
- Fuse.js

## Setup
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Preview Build
```bash
npm run preview
```

## Netlify Deploy
- Build command: `npm run build`
- Publish directory: `dist`

## Assets
- Profile image: `public/assets/profile.png`
- Resume pdf: `public/assets/Azhad_Shahzad_Shaik_Cybersecurity_Engineer_Resume.pdf`

## Features
- Split hero premium layout
- Sticky navbar
- Command palette (`Ctrl/Cmd+K`)
- Fuse.js fuzzy search with URL sync (`?q=`)
- Card/Table view toggle persisted in localStorage
- Certifications category filters + show more toggle
- Mobile experience accordion
- Motion with reduced-motion support
